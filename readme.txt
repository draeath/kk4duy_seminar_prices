Seminar Prices v1.00

https://www.nexusmods.com/x4foundations/mods/769 (as an optional file)
https://steamcommunity.com/sharedfiles/filedetails/?id=2774049353

-----------------------------

Modifies seminar prices to make them more expensive across-the-board. It also disables selling seminars to traders, due to the higher prices unbalancing seminar mission rewards. A sufficiently high faction reputation overrides this.

Intended to be used alongside my "Trader Seminars" mod, which makes them all purchasable at traders.

Save compatible.

-----------------------------

Modifiziert die Seminarpreise, um sie durchgängig zu verteuern. Es verhindert auch den Verkauf von Seminaren an Händler, da die höheren Preise die Belohnungen für Seminarmissionen aus dem Gleichgewicht bringen. Eine ausreichend hohe Fraktionsreputation setzt dies außer Kraft.

Soll zusammen mit meinem "Händlerseminare"-Mod verwendet werden, der sie alle bei Händlern käuflich macht.

Kompatibel speichern.
